# Sales Data Analysis using SQL

## Project Overview
Analyzed retail sales dataset to identify business insights including revenue trends, profit distribution, and top-performing product categories.

## Dataset
Superstore sales dataset containing:
- Orders
- Categories
- Sub-categories
- Sales
- Profit
- Region

## Business Questions Solved
1. Total company sales
2. Sales by category
3. Profit by region
4. Top selling sub-categories
5. High revenue generating products

## SQL Skills Used
- Aggregations (SUM, COUNT)
- GROUP BY
- ORDER BY
- LIMIT
- Business KPI calculations

## Insights
- Identified highest revenue generating category
- Found most profitable region
- Determined top selling products
